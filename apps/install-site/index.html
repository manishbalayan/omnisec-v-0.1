What You Get

🕵️

Agent Discovery

Automatically discovers AI agents running on your machine — Claude Code, Python processes, Node.js, Docker containers, and system processes.

❤️

Health Monitoring

Continuous health checks with crash detection. When a process dies, OmniSec detects it immediately and generates events.

🔄

Auto-Restart

Failed agents are automatically restarted with configurable backoff. Recovery events are logged and persisted.

🛡️

Security Monitoring

Real-time behavioral analysis — traffic anomaly detection, new destination tracking, fingerprint drift, and correlation engine.

📊

Dashboard

Full web dashboard with agent overview, event timeline, reliability metrics, and security operations center view.

💾

Persistent Storage

All events, incidents, and metrics are stored in PostgreSQL with automatic volume persistence across restarts.
